# git commands

## 🚀 Basic comandos
| Comando | Descripcion |
| :--- | :--- |
| `git add .` | add all file modifications. |
| `git commit -m "message"` | commit and message |
| `git branch -M main` | change master to main name |
| `git remote add origin https://github.com/` | connect to the remote repository |
| `git push -u origin main` | push changes|
